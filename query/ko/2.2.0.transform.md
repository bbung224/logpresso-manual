## 2.2. 데이터 가공


* parse: 파서 기반 구문 분석
* parsekv: 키/값 구문 분석
* parsejson: JSON 구문 분석
* eval: 표현식 평가
* evalc: 쿼리 컨텍스트 변수 설정
* explode: 행 분할
* fields: 필드 선택
* limit: 쿼리 결과 페이징
* lookup: 매핑테이블 조회
* rename: 필드 이름 변경
* rex: 정규표현식 기반 필드 추출
* search: 조건식 필터링
* signature: 시그니처 추출
* sort: 정렬
* stats: 통계
* timechart: 시간대별 통계
* join: 조인
* boxplot: 박스플롯
